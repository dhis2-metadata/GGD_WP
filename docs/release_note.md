# Release Note { #ggd-wp-release-note }

## 1.0.0

Initial release of the WorldPop Global2 metadata package.

Content:
* `Population (WorldPop Global2)` Data Set and matching Data Element Group
* Data Elements:
  * `Population totals (WorldPop Global2)`
  * `Population sex/age groups (WorldPop Global2)` with accompanying Category Options, Categories, Category Combos and Category Option Combos
* User Groups:
  * `WorldPop - Admin` to import data 
  * `WorldPop - Access` to access data in analytics apps
